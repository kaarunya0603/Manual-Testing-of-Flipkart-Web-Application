### Manual-Testing-of-Flipkart-Web-Application
Manual testing project on Flipkart covering test cases and bug reports for key e-commerce modules

### Flipkart Manual Testing Project

## Project Description
This project focuses on manual testing of a live e-commerce web application (Flipkart) for learning and portfolio purposes.
The goal is to design test cases, execute them, and report defects in a professional QA format.

### Modules Tested
1. Login
2. Search
3. Search with Filters and Sorting
4. Product Details Page (PDP)
5. Add to Cart & Cart Page

# Types of Testing Performed
 1. Functional Testing
 2. UI/UX Testing
 3. Exploratory Testing

## Test Artifacts
 1. Test cases written in Excel/Google Sheets
 2. Executed test cases with Pass/Fail status
 3. Defect (Bug) reports with severity and priority

## Tools Used
 1. Google Sheets / Microsoft Excel
 2. Web Browser (Google Chrome)
 3. GitHub

## Learning Outcome
 1. Hands-on experience in writing test cases
 2. Understanding of real-time bug reporting
 3. Practical exposure to e-commerce application testing
